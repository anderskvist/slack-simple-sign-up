# slack-simple-sign-up
A simple sign up slash command for Slack

# Functions
## Create an event
/event create NameOfEvent 2015-12-17 19:00 [2015-12-16 16:00]

## List events
/event list

## Signup for an event
/event attend NameOfEvent [1 [message]]

## See status for an event
/event status NameOfEvent
