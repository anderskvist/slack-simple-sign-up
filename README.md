# slack-simple-sign-up
A simple sign up slash command for Slack

# Functions
## Create an event
/signup create 2015-12-17 NameOfEvent [2015-12-16 [16:00]]

## Signup for an event
/signup NameOfEvent 1 [message]

## See status for an event
/signup status NameOfEvent
